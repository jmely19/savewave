Introducción a las fórmulas en Excel

Espacio para imagen: Barra de fórmulas de Excel con una función SUMA

Recomendación: Captura de pantalla mostrando la escritura de una fórmula

¿Qué es una fórmula?

  • Una expresión que realiza cálculos con los datos de tu hoja de cálculo.
  • Siempre comienza con el signo = (igual).
  • Puede contener números, referencias a celdas, operadores y funciones.
  • Ejemplo: =A1+B1 suma los valores de las celdas A1 y B1.

Operadores aritméticos básicos:

Operador Función Ejemplo Resultado
+ Suma =A1+B1 Suma los valores de A1 y B1
- Resta =A1-B1 Resta B1 de A1
* Multiplicación =A1*B1 Multiplica A1 por B1
/ División =A1/B1 Divide A1 entre B1

Ejemplo práctico:

Para calcular el total de ventas diarias donde B2, C2 y D2 son las ventas por turno:

=B2+C2+D2

Referencias de celdas

Espacio para imagen: Ejemplo de referencias relativas y absolutas

Recomendación: Captura mostrando cómo cambian las referencias al copiar fórmulas

Referencias relativas:

  • Se ajustan automáticamente al copiar la fórmula a otras celdas.
  • Ejemplo: Si copias =A1+B1 de la fila 1 a la fila 2, se convertirá en =A2+B2.

Referencias absolutas (con $):

  • Fijan la columna o fila al copiarse.
  • Ejemplo: =$A$1+B1 siempre usará la celda A1, aunque la fórmula se copie a otra fila.

¿Cuándo usarlas?

Absolutas

Para constantes como tasas de cambio, factores fijos, etc.

=$B$1*A2

Donde $B$1 contiene una tasa fija que se aplica a diferentes valores en la columna A.

Relativas

Para cálculos repetitivos que deben adaptarse a cada fila/columna.

=A2*B2

Al copiarse a la fila 3, se convierte en =A3*B3, etc.

Funciones básicas de Excel

Espacio para imagen: Menú de funciones en Excel o ejemplo de uso

Recomendación: Captura mostrando el asistente de funciones o autosuma
Función Descripción Sintaxis Ejemplo
SUMA Suma un rango de celdas =SUMA(rango) =SUMA(A1:A10)
PROMEDIO Calcula la media de valores =PROMEDIO(rango) =PROMEDIO(B2:B20)
CONTAR Cuenta celdas con números =CONTAR(rango) =CONTAR(C1:C100)
MÁX Devuelve el valor más alto =MÁX(rango) =MÁX(D2:D50)
MÍN Devuelve el valor más bajo =MÍN(rango) =MÍN(E2:E50)

Aplicación real:

Calificaciones

=PROMEDIO(F2:F30)

Calcula la calificación media de un grupo de estudiantes.

Ventas

=MÁX(G2:G100)

Identifica el día con más ventas en un trimestre.

Uso del Autosuma

Espacio para imagen: Botón de autosuma en Excel

Recomendación: Captura mostrando el menú desplegable de autosuma

¿Cómo funciona?

  1. Selecciona la celda donde quieres el resultado (normalmente debajo de una columna de números o al final de una fila).
  2. Haz clic en el botón Autosuma (Σ) en la pestaña Inicio.
  3. Excel sugerirá un rango. Verifica que sea correcto.
  4. Presiona Enter para confirmar.

Ventajas:

  • Rápido y eficiente para sumar columnas/filas completas.
  • También funciona con otras funciones como PROMEDIO, CONTAR, MÁX, MÍN (haciendo clic en la flecha junto a Σ).
  • Detecta automáticamente el rango a calcular basándose en los datos adyacentes.

Barra de fórmulas y asistente de funciones

Barra de fórmulas

Muestra y edita el contenido de la celda activa.

Es especialmente útil para verificar y editar fórmulas largas y complejas.

Asistente de funciones (fx)

Ayuda a insertar funciones correctamente.

  1. Haz clic en fx junto a la barra de fórmulas.
  2. Elige una función (ej: CONTAR.SI).
  3. Completa los argumentos guiados por Excel.

Ejemplo práctico completo

Vamos a crear una tabla de calificaciones de estudiantes y aplicar las funciones aprendidas:

Estudiante Nota 1 Nota 2 Nota 3 Total Promedio
Ana 85 90 78 =SUMA(B2:D2) =PROMEDIO(B2:D2)
Carlos 92 88 95 =SUMA(B3:D3) =PROMEDIO(B3:D3)
María 76 82 80 =SUMA(B4:D4) =PROMEDIO(B4:D4)
Estadísticas =SUMA(E2:E4)
Promedio del grupo =PROMEDIO(F2:F4)
Nota más alta =MÁX(F2:F4)
Nota más baja =MÍN(F2:F4)

💡 Consejo profesional

Usa nombres descriptivos para tus rangos de datos. En lugar de usar =SUMA(A1:A10), puedes nombrar el rango "Ventas_Trimestre1" y usar =SUMA(Ventas_Trimestre1), lo que hace tus fórmulas más legibles y fáciles de mantener.

Actividades prácticas

Creación de Hoja de Gastos

Mini-Proyecto: Calificaciones

Calcular el promedio de notas de un grupo de estudiantes y identificar la nota más alta y más baja usando las funciones aprendidas.

Tabla de calificaciones

Instrucciones para la actividad de gastos:

  1. Crea una tabla con las columnas: Fecha, Descripción, Categoría, Monto.
  2. Ingresa al menos 10 gastos de diferentes categorías (comida, transporte, entretenimiento, etc.).
  3. Utiliza la función =SUMA() para calcular el total de gastos.
  4. Emplea la función =PROMEDIO() para encontrar el gasto promedio.
  5. Usa =MÁX() y =MÍN() para identificar el gasto mayor y menor.
  6. Por último, aplica =CONTAR() para saber cuántos gastos registraste.
