FUNCIÓN CONCATENAR

La función CONCATENAR une dos o más cadenas de texto en una sola. Es muy útil para combinar nombres y apellidos, direcciones, o cualquier información que esté separada en diferentes celdas.

=CONCATENAR(texto1, [texto2], ...)

FUNCIÓN SI

La función SI realiza una prueba lógica y devuelve un valor si la prueba es verdadera y otro valor si es falsa. Es ideal para crear condiciones en tus hojas de cálculo.

=SI(prueba_lógica, valor_si_verdadero, [valor_si_falso])
Ejemplo visual: Diagrama de flujo que muestra cómo funciona la condición SI

FUNCIÓN BUSCARV

BUSCARV busca un valor en la primera columna de un rango y devuelve un valor en la misma fila desde la columna que especifiques. Es perfecta para buscar información en tablas grandes.

=BUSCARV(valor_buscado, rango_tabla, número_indice_columna, [busqueda_aproximada])
Ejemplo visual: Tabla de productos con BUSCARV encontrando el precio de un producto específico

FUNCIÓN SI.CONJUNTO

La función SI.CONJUNTO evalúa múltiples condiciones y devuelve el valor correspondiente a la primera condición verdadera. Es muy útil cuando tienes más de dos condiciones posibles.

=SI.CONJUNTO(prueba1, valor1, [prueba2, valor2], ...)
Ejemplo visual: Múltiples condiciones siendo evaluadas en secuencia

FUNCIÓN ENCONTRAR

La función ENCONTRAR devuelve la posición de un texto dentro de otro texto. Es sensible a mayúsculas y minúsculas.

=ENCONTRAR(texto_buscado, texto_donde_buscar, [num_inicio])
